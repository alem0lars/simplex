    Simplex

    Introduction

    Simplex is a optimization software for linear problems. It aims to be as much generic as possible to be able to substitute computing algorithms on-demand, dynamically.

    It is also instructive because of its design principles.

    Problem examples

    • Example1 min f(x) = -x1 -x2 s.t. 6x1 + 4x2 <= 24 3x1 - 2x2 <= 6

    • Example2 min f(x) = x1 + x3 s.t. x1 + 2x2 <= 5 x2 + 2x3 = 6
